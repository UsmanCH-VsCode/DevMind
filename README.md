# L-S Auth Flutter
A complete Login / Sign-Up Auth Architecture using Riverpod, Google Sign-in, Firebase and FireStore

## Login/SignUp Built
- Authentication flow (Login, Sign Up, Logout, Forgot Password)
- Firebase Authentication integration
- Google Sign-In
- Riverpod state management (AsyncNotifier, StreamProvider)
- Auth Controller for handling auth actions
- Firebase Error Mapper for user-friendly error messages
- Splash Screen with auth state detection
- Protected routing (auto redirect based on auth state)
- Login/Sign Up UI with form validation
- Password visibility toggle
- Clean Architecture (data, domain, presentation layers)
